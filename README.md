# nuxt3-app-starter

A Nuxt (3) app starter!

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on http://localhost:3000

```bash
$ yarn dev
```

## Production

Build the application for production:

```bash
$ yarn build:prod

# or ft environment
$ yarn build:ft
```

Locally preview production build:

```bash
$ yarn preview
```

Start the node server:

``` bash
$ yarn start
```
